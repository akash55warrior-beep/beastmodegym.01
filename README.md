# BEASTMODE Gym

Modern gym website for BEASTMODE Gym.

## Features

- Responsive design
- Dark aesthetic theme
- Training programs section
- Gallery section
- Instagram integration
- Mobile friendly

## Deployment

Upload the repository to GitHub and deploy using Vercel.

Required file structure:

```
project/
│
├── index.html
└── README.md
```
